前言：如果你所在地区**不翻墙的情况下无法打开**github.com网站，那么通过**修改电脑本地hosts文件**就可以正常打开github.com，或者通过github镜像网站来访问github。

具体方法：

https://ineo6.github.io/hosts

https://github.com/ineo6/hosts

原理：https://zhuanlan.zhihu.com/p/107334179
